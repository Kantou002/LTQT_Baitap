YÊU CẦU HỆ THỐNG:
  - Windows 10 / 11 (64-bit)
  - .NET 8.0 Windows Desktop Runtime
  - SQL Server Express (hoặc SQL Server)

CÀI ĐẶT:
  1. Cài .NET 8.0 Desktop Runtime từ:
     https://dotnet.microsoft.com/download/dotnet/8.0
  2. Cài SQL Server Express từ:
     https://www.microsoft.com/en-us/sql-server/sql-server-downloads
  3. Chạy Setup_QuanLyBanHang_v2.0.exe và làm theo hướng dẫn.
  4. Mở SSMS, chạy file QLBH_v2.0.sql để tạo dữ liệu mẫu.

SỬ DỤNG:
  - Xem file Help\HuongDanSuDung.html hoặc nhấn F1 trong phần mềm.
  - Phím tắt: Ctrl+F1 mở hướng dẫn sử dụng.
